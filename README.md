🚀 Guía Técnica: Migración de VMware Workstation a Proxmox VE 8.4

Este repositorio contiene la documentación técnica y el análisis de un laboratorio de migración de activos virtuales realizado durante mi periodo de formación práctica en BeCall Group León.

📝 Resumen del Proyecto

El objetivo principal fue migrar una infraestructura heterogénea desde un hipervisor de Tipo 2 (VMware) hacia una infraestructura nativa de Tipo 1 (Proxmox VE 8.4), garantizando la integridad de los datos y la optimización del hardware físico (Servidor HP).

🛠️ Tecnologías y Herramientas

Hipervisores: VMware Workstation ➡️ Proxmox VE 8.4.

Sistemas Operativos: Windows 10, Ubuntu Desktop/Server, TinyCore Linux.

Protocolos y Servicios: SCP, SSH, Cloud-Init, VirtIO.

Documentación: Maquetada profesionalmente en LaTeX.

📂 Archivos en este Repositorio

Manual_Proxmox_Bruno.pdf: Guía técnica completa en formato PDF.

main.tex: Código fuente en LaTeX para la generación del documento.

/images: Capturas de pantalla utilizadas en la documentación.

💡 Retos Técnicos Superados

Gestión de Drivers: Configuración de VirtIO para sistemas Windows.

Persistencia en TinyCore: Resolución de fallos de arranque mediante re-mapeo de buses (SATA).

Troubleshooting: Resolución de incidencias de permisos SSH y espacio en disco durante transferencias SCP.
